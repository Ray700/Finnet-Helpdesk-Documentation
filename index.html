1. Introduction

Welcome to the Finnet Helpdesk User Guide. This guide provides step-by-step instructions on how to use the Helpdesk system powered by Odoo 17 to manage customer support tickets effectively.

2. Accessing Helpdesk

  • Login to your Odoo system using your assigned credentials.
  • Navigate to the Helpdesk module from the main dashboard.
  • You will be directed to the Helpdesk overview dashboard.

3. Creating and Managing Tickets

  • Click on Create to open a new support ticket.
  • Fill in the necessary details such as Subject, Assigned Team, and Description.
  • Set Priority (Low, Medium, High, Urgent) and add tags if necessary.
  • Click Save to register the ticket.
